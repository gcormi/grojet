📖 Mode d'emploi

Cet outil est conçu pour vous faciliter la surveillance de l'épreuve. Voici comment l'utiliser :

  • 1. Préparez l'épreuve : Utilisez le menu déroulant en haut à gauche pour sélectionner l'épreuve du jour. Toutes les informations s'ajusteront automatiquement.
  • 2. Ajustez les règles : Le bouton "📋 Règles" vous permet d'afficher ou masquer certaines consignes (smartphones, montres connectées...) selon l'épreuve.
  • 3. Donnez le Top Départ : Au moment de la distribution des sujets, saisissez l'heure exacte dans la case "Début".
  • 4. Tiers-Temps : Cliquez sur le bouton "⏱️ Afficher 1/3 Tiers-Temps" pour faire apparaître les horaires aménagés spécifiques.
  • 5. N° du Candidat : Cliquez directement sur la case "N° DU CANDIDAT" sur la copie virtuelle pour afficher "NE PAS REMPLIR" (ex: DNB Blanc).
  • 6. Plein Écran : Cliquez sur le bouton "🔲 Plein Écran" pour masquer les menus de votre ordinateur lors de la projection.
