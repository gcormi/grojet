Interface IA 🤖

Configuration de l'IA

Choisissez votre fournisseur d'IA et entrez votre clé API. Elle sera sauvegardée dans votre navigateur.

⚠️ Votre clé est stockée localement. Ne l'utilisez pas sur un appareil public.

⚠️ **Note :** Les prompts longs (comme la génération d'un cours complet) consommeront plus de tokens et peuvent être bloqués par les limites de votre clé API (surtout les clés d'essai).
